# 常见问题

## 安装与下载

### 下载的文件名代表什么？

MaaKEDR 的发布压缩包命名格式为：

```
MaaKEDR-<系统>-<架构>-<版本>-<前端>.<扩展名>
```

例如 `MaaKEDR-win-x86_64-v1.0.2-MFAA.zip` 表示：

- **系统**: `win`（Windows）
- **架构**: `x86_64`（64 位 Intel/AMD）
- **版本**: `v1.0.2`
- **前端**: `MFAA`（图形界面，需 .NET）
- **扩展名**: `zip`

### 下载速度慢怎么办？

GitHub 下载速度受网络环境影响，可尝试：

- 使用 [ghproxy](https://ghproxy.com/) 等代理加速
- 在 MaaKEDR 交流群内获取网盘备份（见 `CONTACT` 文件）

## 运行崩溃

### 启动时提示缺少 DLL（如 `VCRUNTIME140.dll`）

未安装 VCRedist。请下载安装 [vc_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)。

### MFAA 启动无反应或报 .NET 相关错误

未安装 .NET 桌面运行时。请前往 [.NET 下载页面](https://dotnet.microsoft.com/download/dotnet) 下载并安装对应版本。

### 程序闪退，没有错误提示

检查 `debug/maafw.log` 查看错误日志。常见原因：

- 路径包含中文字符或空格 → 移至纯英文路径
- 解压不完整 → 重新解压
- 杀毒软件误删文件 → 添加排除项

### MXU 版本启动卡顿或黑屏

确保显卡驱动已更新。在 MXU 设置中尝试切换渲染模式。

## 连接问题

### 无法自动检测到模拟器

- 确认模拟器 ADB 调试已开启
- 确认只启动了一个模拟器实例
- 尝试点击"刷新"按钮
- 确认模拟器分辨率设置为 1280x720（横屏）

### 手动连接如何配置？

在连接设置中，点击"自定义"可手动输入 ADB 地址。各模拟器默认端口：

| 模拟器       | 默认 ADB 地址     |
| :----------- | :---------------- |
| MuMu 12      | `127.0.0.1:16384` |
| BlueStacks 5 | `127.0.0.1:5555`  |
| LDPlayer 9   | `127.0.0.1:5555`  |
| 夜神模拟器   | `127.0.0.1:62001` |

### 连接成功但任务无法执行

- 检查游戏是否已安装并更新到最新版本
- 确认已在 MaaKEDR 中选择了正确的资源类型（官服/B 服/TapTap 服）
- 检查 `debug/` 目录下的截图，确认识别图像是否正确

## 任务执行

### Agent 启动超时

首次运行时 MaaKEDR 需要安装 Python 依赖，可能耗时较长。请耐心等待，不要点击停止。若反复超时：

- 检查网络连接是否正常
- 检查 `config/pip_config.json` 中的镜像源是否可用

### 任务未按预期执行

- 确认任务已勾选启用
- 阅读任务的"任务说明"，按要求配置选项
- 确保游戏处于正确的界面（如主菜单或对应功能页面）
- 查看 `debug/maafw.log` 日志文件定位问题

### 资源刷取选项中的关卡列表不全

当前版本仅支持部分预设关卡。如需其他关卡，可参考开发文档中的 Pipeline 指南自行配置。

## 其他

### 如何查看日志？

运行日志位于 `debug/maafw.log`，截图位于 `debug/` 目录下。提交问题反馈时请一并提供这些文件。

### 如何更新 MaaKEDR？

MaaKEDR 支持软件内更新。在软件设置中将更新源选择为 **GitHub**，点击检查更新即可自动下载并安装最新版本。

如需手动更新：下载最新版本压缩包，解压到新目录，将旧版本的 `config/` 目录复制到新目录即可保留配置。

# 功能介绍

MaaKEDR 提供以下自动化任务，帮助你更高效地完成《雪松》的日常操作。

---

## 启动游戏

自动启动游戏并进入主界面。

**功能：**

- 启动游戏进程
- 点击"开始游戏"
- 处理每日登录奖励弹窗
- 检测主界面并完成进入

无需额外配置，勾选启用即可。

---

## 领取奖励

一键领取各类奖励。支持按类别独立开关。

**选项：**

| 选项                   | 类型 | 说明                           |
| :--------------------- | :--- | :----------------------------- |
| 派遣任务领取           | 开关 | 是否领取并再次派遣资源派遣任务 |
| 每日/每周/军旅成就奖励 | 开关 | 是否领取每日/每周/军旅成就奖励 |
| 战令通行证奖励         | 开关 | 是否领取战令通行证奖励         |
| 邮箱奖励               | 开关 | 是否领取邮箱奖励               |

### 派遣任务领取

开启后，MaaKEDR 会：

1. 进入派遣界面
2. 领取已完成派遣的奖励
3. 如开启 **再次派遣**，选择再次派遣
4. 关闭派遣界面

### 每日/每周/军旅成就奖励

每日签到、周常任务和军旅成就的奖励领取。

### 战令通行证奖励

自动领取战令通行证的已解锁奖励。

### 邮箱奖励

自动领取邮件附件。

---

## 资源刷取

自动刷取资源关卡，支持体力管理。

**选项：**

| 选项     | 类型 | 说明                |
| :------- | :--- | :------------------ |
| 战斗模式 | 选择 | 单次战斗 / 清空体力 |
| 刷取板块 | 选择 | 资源收集 / 技能演练 |
| 资源类型 | 选择 | 见下方              |
| 战斗次数 | 选择 | 1–6 次 / 最大       |

### 战斗模式

- **单次战斗**：按设定的战斗次数刷取，完成后停止
- **清空体力**：持续刷取直至体力耗尽，适合睡前挂机

### 资源收集

| 资源类型     | 说明              |
| :----------- | :---------------- |
| 特别军费行动 | 军费/货币关卡     |
| 作战体能训练 | 经验/升级材料关卡 |
| 兵种能力评级 | 兵种晋升材料关卡  |
| 载具对抗演练 | 载具相关材料关卡  |

关卡数量因类型而异，可在选项中具体选择：

- 特别军费行动：1-1 ~ 1-5
- 作战体能训练：2-1 ~ 2-4
- 兵种能力评级：3-1 ~ 3-4
- 载具对抗演练：4-1 ~ 4-5

### 技能演练

| 技能类型     | 说明             |
| :----------- | :--------------- |
| 基础技能演练 | 基础技能升级材料 |
| 专业技能演练 | 高级技能升级材料 |

---

## 玩家对战

自动进行玩家对战（PVP），支持多场次循环。

**选项：**

| 选项     | 类型 | 说明   |
| :------- | :--- | :----- |
| 战斗次数 | 选择 | 1–5 次 |

MaaKEDR 会自动：

1. 进入玩家对战界面
2. 匹配并开始战斗
3. 识别战斗结果
4. 重复直至达到设定次数

> [!NOTE]
>
> PVP 战斗结果受游戏内阵容和练度影响，MaaKEDR 仅自动化操作流程，不保证胜利。

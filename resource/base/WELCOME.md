# MaaKEDR

基于 MaaFramework 的《雪松》(KEDR) 自动化助手。

## 功能

- **启动游戏** — 自动启动并进入游戏主界面
- **领取奖励** — 每日/每周/军旅成就奖励、战令通行证、邮箱、派遣任务
- **资源刷取** — 支持资源收集/技能演练，普通模式与清空体力模式

## 使用说明

1. 连接 Android 模拟器/设备（ADB，1280x720）
2. 选择任务并配置选项
3. 点击启动开始运行

# MaaKEDR 长期公告

![LOGO](maakedr-logo_128x128.png)

基于 MaaFramework 的《雪松》(KEDR) 小助手。图像识别 + 模拟控制，解放双手！

## 功能

- **启动游戏** — 自动启动并进入游戏主界面
- **领取奖励** — 每日/每周/军旅成就奖励、战令通行证、邮箱、派遣任务
- **资源刷取** — 资源收集与技能演练，支持普通模式与清空体力模式

## 使用说明

1. 打开 Android 模拟器，确保 ADB 可连接（分辨率 1280x720）
2. 启动 MaaKEDR，点击右上角刷新连接模拟器
3. 在任务列表勾选需要执行的任务
4. 配置任务选项，点击启动开始运行

# MaaKEDR 长期公告

![LOGO](https://raw.githubusercontent.com/APPLe-DF/MaaKEDR/master/resource/announcement/maakedr-logo_128x128.png)

基于 MaaFramework 的《雪松》(KEDR) 小助手。图像识别 + 模拟控制，解放双手！

> **重要提示**：
>
> 本项目 GitHub 链接：[@APPLe-DF/MaaKEDR](https://github.com/APPLe-DF/MaaKEDR)  
> 软件本体完全免费，请勿购买任何付费版本，若您有付费购买请立即退款并举报！

## 功能

- **启动游戏** — 自动启动并进入游戏主界面
- **领取奖励** — 派遣任务、每日/每周/军旅成就奖励、战令通行证、邮箱
- **资源刷取** — 资源收集与技能演练，支持单次战斗与清空体力模式
- **玩家对战** — 自动进行 PVP 对战，识别积分排名变化，支持多场次循环

## 使用说明

1. 打开 Android 模拟器，确保 ADB 可连接（分辨率 大于等于720p的16:9分辨率，推荐720p）
2. 启动 MaaKEDR，点击右上角刷新连接模拟器
3. 在任务列表勾选需要执行的任务
4. 配置任务选项，点击启动开始运行

## 项目状态

本项目在完成清每日流程功能后已转为**维护性更新模式**，不再主动添加新内容，仅进行 bug 修复和必要的适配更新。如需更完善的功能，建议关注同类开源项目 [MaaAssistantKedrgame (MAK)](https://github.com/Hollow-YK/MaaAssistantKedrgame)。
